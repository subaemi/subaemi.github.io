Основные типы данных в Python

В Python, как и в большинстве языков программирования, существуют различные типы данных, которые используются для хранения и обработки информации. Давайте рассмотрим основные из них:

Тип данных Описание Пример
int Целые числа (без дробной части) x = 10
float Числа с плавающей точкой (с дробной частью) y = 3.14
str Строки (текст) name = "Python"
bool Логический тип (True или False) is_true = True
list Список (упорядоченная коллекция элементов) numbers = [1, 2, 3]
tuple Кортеж (неизменяемая упорядоченная коллекция элементов) coordinates = (10, 20)
dict Словарь (неупорядоченная коллекция пар "ключ-значение") person = {"name": "Alice", "age": 30}
set Множество (неупорядоченная коллекция уникальных элементов) unique_numbers = {1, 2, 3}

Примеры использования

Давайте посмотрим на несколько примеров использования этих типов данных:


# Целое число
age = 30

# Число с плавающей точкой
price = 99.99

# Строка
message = "Привет, мир!"

# Логическое значение
is_adult = True

#Список
fruits = ["яблоко", "банан", "апельсин"]

# Кортеж
point = (10, 20)

# Словарь
student = {"name": "Bob", "age": 25, "grade": "A"}

# Множество
unique_letters = {"a", "b", "c"}

print(age, price, message, is_adult, fruits, point, student, unique_letters)
