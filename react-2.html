JSX: Синтаксис React

JSX (JavaScript XML) — это синтаксическое расширение JavaScript, используемое в React для описания структуры пользовательского интерфейса. Он позволяет писать HTML-подобный код внутри JavaScript, что делает код более читаемым и понятным.

JSX-код компилируется в обычный JavaScript перед выполнением, поэтому вам не нужно беспокоиться о совместимости браузеров.

Компоненты: строительные блоки React

Компоненты — это независимые, многократно используемые блоки кода, которые составляют пользовательский интерфейс в React. Они могут быть функциональными или классовыми.
