Веб-разработка на React - Урок 3: Props

Props: Передача данных в компоненты

Props (properties — свойства) — это механизм передачи данных из родительского компонента в дочерний. Они являются способом обмена информацией между компонентами в React. Props передаются в качестве атрибутов в JSX.

Props являются неизменяемыми (immutable) внутри дочернего компонента. Это означает, что дочерний компонент не может изменять значения props.

Пример:


function Welcome(props) {
  return 
Привет, {props.name}!
;

function App() {

В этом примере, компонент `App` передает props `name` компоненту `Welcome`. Компонент `Welcome` отображает приветствие с переданным именем.

Типы данных props:

Props могут принимать различные типы данных, включая строки, числа, булевы значения, массивы и объекты.

Проверка типов props (propTypes):

(Для более надежного кода, вы можете использовать `propTypes` для проверки типов переданных props)


                Welcome.propTypes = {
                name: PropTypes.string.isRequired,
                };
